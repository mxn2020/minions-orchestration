# Welcome to Minions Orchestration

We're excited to introduce **Minions Orchestration** — Clawspace registry, toolbox-to-agent mappings, and fleet configuration.

## What's Included

- **TypeScript SDK**: `npm install @minions-orchestration/sdk`
- **Python SDK**: `pip install minions-orchestration`
- **CLI**: `npm install -g @minions-orchestration/cli`

## Getting Started

Check out the [documentation](https://orchestration.minions.help) for a full guide.
