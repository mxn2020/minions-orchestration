---
title: Installation
description: Install Minions Orchestration packages
---

## TypeScript / Node.js

```bash
npm install @minions-orchestration/sdk minions-sdk
```

## Python

```bash
pip install minions-orchestration
```

## CLI

```bash
npm install -g @minions-orchestration/cli
```

## Requirements

- **Node.js** 20 or later
- **Python** 3.11 or later (for Python SDK)
