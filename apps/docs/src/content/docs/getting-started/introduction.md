# Minions Orchestration

Clawspace registry, toolbox-to-agent mappings, and fleet configuration

## Overview

This project is part of the [Minions ecosystem](https://github.com/mxn2020/minions) and provides:

- **TypeScript SDK** (`@minions-orchestration/sdk`) — Core library for Node.js
- **Python SDK** (`minions-orchestration`) — Python bindings
- **CLI Tool** (`@minions-orchestration/cli`) — Command-line interface
- **Web App** — Interactive playground at [orchestration.minions.wtf](https://orchestration.minions.wtf)
